# Sistema Sabor Caseiro - Scripts SQL

## Descrição
Este repositório contém os scripts SQL para o sistema de gestão do restaurante Sabor Caseiro.

## Estrutura
- `criacao_tabelas.sql`: Script DDL para criação do banco e tabelas
- `insercao_dados.sql`: Script DML para inserção de dados iniciais
- `consultas.sql`: Diversas consultas SELECT
- `atualizacao.sql`: Comandos UPDATE
- `exclusao.sql`: Comandos DELETE

## Como Executar
1. Execute `criacao_tabelas.sql` primeiro
2. Execute `insercao_dados.sql`
3. Execute os demais scripts conforme necessidade

## Tecnologias
- MySQL 8.0+
- MySQL Workbench
